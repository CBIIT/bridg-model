The target site is often implicit in the definition of the observation itself.<br /></p><p><br /></p><p>Sources that capture anatomic site and laterality separately should map both to this attribute. For implementation models based on BRIDG where site and laterality are captured separately, you may wish to capture both concepts as a post-coordinated code structure or as multiple code repetitions<br /></p>
