When the DocumentAuthor is an author for a StudyProtocolDocumentVersion, the DocumentAuthor must be a function performed by a HealthcareProvider or ResearchStaff only.<br/>

Details:
A DocumentAuthor is a function performed by one and only one of the following: ResearchStaff, Subject, AssociatedBiologicEntity, HealthcareProvider, or Person.<br/>

Details:
