<p>The character string datatype stands for text data, primarily intended for machine processing (e.g., sorting, querying, indexing, etc.) or direct display. Used for names, symbols, presentation and formal expressions.<br/></p><p><br/></p><p>A ST SHALL have at least one character or else have a nullFlavor<br/></p>
