<p>Specializes COLL<br/></p><p><br/></p><p>Parameter: T : ANY<br/></p><p><br/></p><p>A collection that contains discrete (but not necessarily distinct) values in a defined sequence. Values are also assigned an offset; the first value has the offset of zero.<br/></p><p><br/></p><p>The sequence is an ordered collection of values, but no particular order is inherently associated with the sequence. The meaning of the order of the items SHOULD be defined where a LIST is used. Note that in some cases, the order is fixed ( e.g. HIST), but in other cases, the order is not fixed: only the meaning associated with the order in the instance is defined (e.g. EN, AD).<br/></p>
