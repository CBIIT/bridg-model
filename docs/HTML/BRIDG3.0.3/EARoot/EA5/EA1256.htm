<p>A part of a name or address. Each part is a character string that may be coded, and that also may have a nullFlavor. The string content must always be provided whether a code is provided or not.<br/></p>

Attribute Details
public NullFlavor
  nullFlavor
Notes: If the part is not a proper value, indicates the reason.<br /><br />For further information concerning nullFlavor, see 7.3.3.3.1
public String
  value		 Range:0 to 1
Notes: The actual string value of the part. If no nullFlavor is provided, some content must be present in this attribute.
public String
  code		 Range:0 to 1
Notes: A code assigned to the part by some coding system, if appropriate.
public String
  codeSystem		 Range:0 to 1
Notes: The code system from which the code is taken. <br /><br />The choice of coding system depends on the part type defined in the concrete specializations. The codeSystem SHALL be populated if a code is populated.
public String
  codeSystemVersion		 Range:0 to 1
Notes: The version of the coding system, if required.<br /><br />The codeSystem SHALL be populated if a codeSystemVersion is populated.
public Code
  language
Notes: The human language of the content. Valid codes are taken from the IETF <a href="http://www.ietf.org/rfc/rfc3066.txt" \o "http://www.ietf.org/rfc/rfc3066.txt">RFC 3066</a>. If this attribute is null, the language may be inferred from elsewhere, either from the context or from unicode language tags, for example.<br /><br />While parts may be assigned a language, the meaning of the part is not dependent on the language, and applications SHALL not be required to indicate the linguistic origin of any name or address part.
