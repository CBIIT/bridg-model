DEFINITION:<br /></p><p>The unique identification of a genetic variation in a specified context.<br /></p><p><br /></p><p>EXAMPLE(S):<br /></p><p>An identifier in dbSNP, a mutation identifier in COSMIC, or a variant identifier in OMIM <br /></p><p><br /></p><p>OTHER NAME(S):<br /></p><p><br /></p><p>NOTE(S):<br /></p><p>Note that dbSNP identifiers specify a location where several nucleotide changes may be located.<br /></p><p><br /></p><p>It is anticipated that at some point in the future a use case will arise for a type code to be added to the identifiers allowing a single database to issue more than one kind of identifier, but it is also hoped that this need may be addressed by a change in the HL7 data type definition for II.  The current pattern of pairing an II attribute with a CD attribute does not support the anticipated use case.<br /></p>
