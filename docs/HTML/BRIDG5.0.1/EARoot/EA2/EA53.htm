DEFINITION:<br/>The completed action of locating or relocating a specimen or specimen collection group from and/or to its storage location.<br/><br/>EXAMPLE(S):<br/>Check-in, check-out, transfer from one location to another<br/><br/>OTHER NAME(S):<br/>Specimen Check-in/Check-out<br/>Specimen Return<br/>Specimen Placement<br/><br/>NOTE(S):<br/>Note that attributes in this class are optional and can be used as needed for the type of transaction being recorded, e.g. check-in can represent the intake of a new specimen using the “to position” attributes only, likewise check-out can represent a specimen being taken out of a container using only the "from position" attributes.<br/>
