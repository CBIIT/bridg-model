DEFINITION:<br/>A set of parameter values necessary to obtain or reconstruct a set of images for an imaging process protocol.<br/><br/>EXAMPLE(S):<br/>A chest Protocol might include three elements:  two localizer CT radiographs (AP and Lateral) and a single helical scan, each of which would be described in separate steps with different parameter values.<br/><br/>A CT Protocol frequently specifies multiple reconstructions. For example, a single helical Acquisition Element may be reconstructed once as thin slices and a second time as thick slices.<br/><br/>OTHER NAME(S):<br/><br/>NOTE(S):<br/>This class combines the common concept for the process protocol element from the DICOM CT Defined Acquisition Technique Module, the DICOM CT Defined Reconstruction Technique Module, DICOM CT Performed Acquisition Technique Module and the DICOM CT Performed Reconstruction Technique Module.<br/>
