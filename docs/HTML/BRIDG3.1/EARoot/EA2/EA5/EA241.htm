Constraint Type Status Detail
Be Participated In By Qualifier Invariant Approved
Associations from Subject (including StudySubject) are valid only for BiologicEntity and Specimen, not for Product.  Associations from ExperimentalUnit are valid for BiologicEntity, BiologicEntityGroup, BiologicEntityPart and Specimen, but not for Product or ProductGroup.
relativeToReference Qualifier Invariant Approved
endRelativeToReference must not logically precede the startRelativeToReference. For example, if startRelativeToReference is "during", endRelativeToReference must not be "before".
Dose Expression Qualifier Invariant Approved
If a dose expression uses parameters, each parameter must be defined as a DefinedExpressionVariableRelationship.localVariableName.
Active Ingredient Qualifier Invariant Approved
activeIngredientDose, periodActiveIngredientDoseTotal must only be used when the PerformedSubstanceAdministration is associated to a StudyAgent that has only one active ingredient. Otherwise, it would be ambiguous to which active ingredient the amounts indicated in these attributes apply.
Dose Qualifier Invariant Approved
productDose, periodProductDoseTotal, treatmentVehicleAmount, activeIngredientDose, periodActiveIngredientDoseTotal must only be used when the PerformedSubstanceAdministration is associated to a single StudyAgent. Otherwise, it would be ambiguous to which agent the amounts indicated in these attributes apply.
Object Type Connection Notes
PerformedProcedure Class Generalization DEFINITION: Each PerformedSubstanceAdministration always specializes one PerformedProcedure. Each PerformedProcedure might be specialized by one PerformedSubstanceAdministration.
