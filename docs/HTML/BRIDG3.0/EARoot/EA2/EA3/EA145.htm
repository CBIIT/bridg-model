<p>An item/element of a randomization book that can be used to assign a subject to a planned arm or portion of an arm in a study. <br/></p><p><br/></p><p>For example, an entry might be mapping Stratum Group to a Treatment Arm.<br/></p>

  • Associations To
Attribute Details
public INT
  positionNumber		 Map:C3PR=BookRandomizationEntry.position
Notes: An integer specifying the value of a numerical sequence for a Stratum Group that should be used to assign a subject to an arm or a portion of an arm. <br /><br />An example would be:<br />StratumGroup#: 0;  Position: 0;  Arm/Portion of Arm: A<br />StratumGroup#: 0;  Position: 1;  Arm/Portion of Arm: B<br /><br />If 2 patients fall in the same Stratum Group i.e. say 0 in the above example, the first patient will be assigned Arm A because the current position would be 0 and the 2nd patient would be assigned Arm B since the current position would be incremented by 1 each time an assignment happens.
public BL
  positionFilledIndicator		 Map:C3PR=StratumGroup.currentPosition
Notes: Specifies whether the position is filled by a subject assignment.
