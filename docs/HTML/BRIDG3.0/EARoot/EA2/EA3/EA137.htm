<p>A relationship between a composite activity and the component activities that comprise it, i.e. parent and child activities.<br/></p><p><br/></p><p>For example, a battery of tests may be composed of multiple routine labs that are always ordered together as a group. Another example is a glucose tolerance test which is comprised of administering glucose and taking multiple timed blood samples which are then tested for glucose.<br/></p><p><br/></p><p>NOTE:  This class helps represent an AND relationship between siblings with the same parent activity.<br/></p>
